# Interactive Ray Tracing Viewer

## Quick Start

Simply open `index_standalone.html` in your web browser - no build step needed!

## Features

- **Real-time 3D navigation** - Orbit around the scene with mouse
- **Live rendering** - See your scene with all mirrors and lights
- **Camera controls** - Rotate, zoom, and pan
- **All scene elements**:
  - Room with walls, floor, ceiling
  - 4 mirrors with reflections
  - Light sources (ceiling light + candles)
  - Interactive controls

## Controls

- **Left mouse drag**: Rotate camera around scene
- **Right mouse drag**: Pan camera
- **Mouse wheel**: Zoom in/out
- **Controls panel**: Adjust camera rotation and distance

## Files

- `index_standalone.html` - Standalone viewer (works directly in browser)
- `index.html` - Full version (requires `npm install` and `npm run dev`)
- `scene.json` - Scene configuration (shared with Python renderer)
